# La Scompagnia Theatre Website

Public website for the Italian theatre company La Scompagnia.

## Tech stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Requirements
- Node.js `24.14.0` (see `.nvmrc`)
- npm `11+`

## Local development
```bash
npm ci
npm run dev
```

## Quality checks
```bash
npm run lint
npm run build
```

## Security and maintenance
- Dependency and CI updates are managed by Dependabot.
- CI checks run on pull requests.
- Security policy: see `SECURITY.md`.
