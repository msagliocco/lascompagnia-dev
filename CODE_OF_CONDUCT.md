# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our standards
- Be respectful and constructive.
- Focus feedback on behavior and code, not people.
- Assume good intent and ask clarifying questions.

## Reporting issues
For conduct concerns, contact the project maintainer privately.

## Enforcement
Unacceptable behavior may result in warnings, temporary bans, or permanent bans from project spaces.
