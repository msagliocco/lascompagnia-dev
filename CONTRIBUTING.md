# Contributing

## Setup
1. Install Node.js `24.14.0` (or use `.nvmrc`).
2. Install dependencies with `npm ci`.
3. Start local dev server with `npm run dev`.

## Before opening a PR
1. Create a focused branch from `main`.
2. Run `npm run lint`.
3. Run `npm run build`.
4. Update docs when behavior or setup changes.

## Pull requests
1. Use small, reviewable PRs with a clear title.
2. Explain user impact and any risk in the PR description.
3. Link related issues.
